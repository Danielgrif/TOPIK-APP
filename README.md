# 🇰🇷 TOPIK Master Pro

!Version !Status !Tech

Современное PWA приложение для подготовки к экзамену TOPIK (Test of Proficiency in Korean) с использованием AI и интервальных повторений.

## 🚀 Возможности

### 📚 Умное изучение слов
*   **SM-2 Алгоритм:** Приложение знает, когда вам нужно повторить слово, чтобы не забыть его.
*   **AI Контент:** Автоматическая генерация примеров, аудио (мужской/женский голос) и изображений.
*   **Hanja:** Изучение иероглифических корней для глубокого понимания.

### 🎮 Геймификация
*   **Режимы:** Спринт, Выживание, Аудирование, Правописание.
*   **Лиги и Рейтинг:** Соревнуйтесь с другими пользователями в еженедельных лигах.
*   **Магазин:** Тратьте заработанные монеты на темы оформления и бонусы.

### 🛠 Технологии
*   **Offline-first:** Работает без интернета. Синхронизация при появлении сети.
*   **Cross-platform:** Устанавливается как приложение на iOS, Android и Desktop.

## 📦 Установка и Запуск

### Требования
*   Node.js 18+
*   Python 3.10+ (для воркера контента)
*   Supabase Project

### Frontend
```bash
npm install
npm run dev
```

### Backend (Content Worker)
Скрипт для генерации контента (аудио, примеры) работает локально или на сервере.
```bash
pip install -r requirements.txt
python scripts/content_worker.py
```

## 📂 Структура проекта

*   `src/core` - Ядро логики (БД, Синхронизация, State).
*   `src/ui` - Логика интерфейса и контроллеры.
*   `src/html` - HTML шаблоны компонентов.
*   `src/css` - Стили.
*   `scripts/` - Python скрипты для управления БД и AI генерации.
*   `supabase/` - Edge Functions (Deno).
